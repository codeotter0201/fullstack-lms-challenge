# 設計規格需求

**目標網站**: https://world.waterballsa.tw/
**最後更新**: 2025-11-22
**基於**: UI 比較分析報告 ([`UI-COMPARISON-REPORT.md`](../UI-COMPARISON-REPORT.md))

本文檔定義水球軟體學院 LMS 前端的設計規格，所有實作應完全符合目標網站設計。

## 🎨 色彩方案

基於目標網站實際測量值。

### 主要顏色

| 用途 | 顏色代碼 | CSS 變數 | 使用場景 |
|------|---------|---------|----------|
| 主背景 | `#171923` | `--background` | 頁面主要背景、Sidebar 背景 |
| 次背景 | `#2D3142` | `--background-secondary` | 卡片背景（如需要） |
| **主色（金色）** | `#FFD700` | `--primary` | **所有按鈕、CTA、高亮元素、卡片邊框** |
| 按鈕文字 | `rgb(23, 25, 35)` | `--button-text` | 金色按鈕上的深色文字 |
| 主文字 | `#F3F4F6` | `--text-primary` | 標題與重要文字 |
| 次文字 | `#A0AEC0` | `--text-secondary` | 描述與輔助文字 |
| 邊框 | `#4A5568` | `--border` | 分隔線、輸入框邊框 |

### 重要設計原則

**統一金色主題 (#FFD700)**：
- ✅ 所有主要按鈕使用金色背景
- ✅ 所有 CTA 元素使用金色
- ✅ 課程卡片邊框使用金色
- ✅ Sidebar 高亮項目使用金色背景
- ✅ 優惠區塊使用金色背景
- ❌ 不要使用其他顏色作為主要行動按鈕

### CSS 變數配置

在 `app/globals.css` 中實作：

```css
:root {
  /* 背景色 - 基於目標網站測量 */
  --background: #171923;
  --background-secondary: #2D3142;

  /* 主色調 - 統一金色 */
  --primary: #FFD700;
  --primary-hover: #E6C200;

  /* 文字色 */
  --text-primary: #F3F4F6;
  --text-secondary: #A0AEC0;
  --button-text: rgb(23, 25, 35);

  /* 邊框 */
  --border: #4B5563;
}

body {
  background-color: var(--background);
  color: var(--text-primary);
}
```

## 🏗️ 佈局架構

基於目標網站實際結構。

### 整體佈局

**重要**：目標網站**沒有獨立的頂部 Navbar**。

```
┌─────────────────────────────────────────┐
│                                         │
│ 左側                主內容區域            │
│ Sidebar           ┌─────────────────┐   │
│ (230px)           │ 選課下拉 + 登入  │   │
│                   ├─────────────────┤   │
│ - Logo            │                 │   │
│                   │                 │   │
│ [主導航]          │  Page Content   │   │
│ - 首頁            │                 │   │
│ - 課程            │                 │   │
│                   │                 │   │
│ [排行榜]          │                 │   │
│ - 排行榜          │                 │   │
│                   │                 │   │
│ [旅程相關]        │                 │   │
│ - 所有單元        │                 │   │
│ - 挑戰地圖        └─────────────────┘   │
│ - SOP 寶典                              │
│                                         │
└─────────────────────────────────────────┘
```

### 左側 Sidebar 規格

**尺寸**：
- 寬度：**230px**（目標網站測量值）
- 背景色：`#171923`（與 body 背景相同）
- 固定定位：`position: fixed; left: 0; top: 0; bottom: 0;`
- 內部 padding：**15px**

**內容區塊**：
1. **Logo 區** (頂部)
   - 水球軟體學院 Logo + 品牌名稱
   - 高度：~64px

2. **主導航** (分組顯示，用空白分隔)
   - **主導航組**：
     - 首頁
     - 課程
   - **排行榜組**：
     - 排行榜
   - **旅程相關組**：
     - 所有單元
     - 挑戰地圖
     - SOP 寶典

3. **導航項目樣式**：
   - 每項高度：~48px
   - **選中狀態**：
     - 背景：金色 (#FFD700)，圓角
     - 文字：深色（與按鈕文字相同）
   - **未選中狀態**：
     - 背景：透明
     - 文字：淺色

**行動版**：
- 隱藏 Sidebar
- 顯示為抽屜式選單（MobileDrawer）
- 漢堡選單觸發

### 主內容區規格

**位置與尺寸**：
- 左邊距：**230px**（Sidebar 寬度）
- 寬度：**970px**（測量值）
- 背景：`#171923`

**頂部功能區**（整合在內容區，非獨立 Navbar）：
- 選課下拉選單（左側）
- 登入按鈕（右側，金色）
- **不是獨立的 sticky header**
- **不包含通知、設定圖標**

## 🔘 按鈕設計

基於目標網站測量值。

### 主要按鈕（Primary）

```css
/* 目標網站規格 */
background: #FFD700;
color: rgb(23, 25, 35);
border-radius: 6px;  /* 注意：不是 8px */
padding: 8px 16px;
font-weight: 500;    /* 注意：不是 400 或 600 */
```

**Tailwind 類別**：
```tsx
className="bg-primary text-[rgb(23,25,35)] rounded-md px-4 py-2 font-medium hover:bg-primary-hover"
```

### 次要按鈕（Secondary）

如果需要次要按鈕：
```css
background: transparent;
color: #FFD700;
border: 1px solid #FFD700;
border-radius: 6px;
padding: 8px 16px;
font-weight: 500;
```

### 按鈕狀態

- **Normal**: 金色背景 + 深色文字
- **Hover**: 稍暗的金色（#E6C200）
- **Active**: 更暗的金色
- **Disabled**: 灰色背景 + 灰色文字

## 📐 字體與排版

### 字體家族

目標網站使用：
```css
font-family: ui-sans-serif, system-ui, sans-serif;
```

建議保持：
```css
font-family: 'Inter', 'Noto Sans TC', ui-sans-serif, system-ui, sans-serif;
```

### 字體大小與權重

| 用途 | 大小 | 權重 | Tailwind 類別 |
|------|------|------|---------------|
| H1 標題 | 48-60px | 700 | `text-5xl font-bold` |
| H2 標題 | 24px | 700 | `text-2xl font-bold` |
| H3 標題（課程卡） | 18-20px | 600 | `text-lg font-semibold` |
| 正文 | 16px | 400 | `text-base font-normal` |
| 小字 | 14px | 400 | `text-sm font-normal` |
| **按鈕文字** | 16px | **500** | `text-base font-medium` |

**重要**：按鈕字重為 **500**（medium），不是 400 或 600。

### 行高

- 標題：`line-height: 1.3`
- 正文：`line-height: 1.6`
- 按鈕：`line-height: 1.5`

## 📱 響應式設計

### 斷點

| 裝置 | 寬度範圍 | Tailwind 前綴 |
|------|---------|--------------|
| Mobile | < 768px | 無前綴 |
| Tablet | 768px - 1023px | `md:` |
| Desktop | ≥ 1024px | `lg:` |

### 行動版調整

**佈局**：
- 隱藏左側 Sidebar → 漢堡選單 + 抽屜（MobileDrawer）
- 選課下拉 + 登入按鈕移至頂部
- 主內容區寬度 100%

**課程卡片**：
- 單欄顯示
- 保持黃色邊框設計
- 縮圖保持 16:9 比例

**觸控優化**：
- 所有互動元素最小 44×44px
- 避免 Hover 效果影響觸控

## 🎯 互動設計

### 懸停效果

**卡片**：
- 懸停時：陰影增強 + 微向上位移
- 過渡：`transition-all duration-300`
- 保持黃色邊框

**按鈕**：
- 懸停時：背景色稍暗（#E6C200）
- 過渡：`transition-colors duration-200`

**連結**：
- 懸停時：顏色變為金色 (#FFD700)
