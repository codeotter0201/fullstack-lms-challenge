## 效能測試

### Lighthouse 評分

- [ ] **Performance (效能)**
  - [ ] 分數 ≥ 90
  - [ ] FCP (First Contentful Paint) < 1.8s
  - [ ] LCP (Largest Contentful Paint) < 2.5s
  - [ ] TBT (Total Blocking Time) < 200ms
  - [ ] CLS (Cumulative Layout Shift) < 0.1

- [ ] **Accessibility (無障礙)**
  - [ ] 分數 ≥ 90
  - [ ] 顏色對比度符合 WCAG AA
  - [ ] 所有互動元素可鍵盤操作
  - [ ] ARIA 屬性正確

- [ ] **Best Practices (最佳實踐)**
  - [ ] 分數 ≥ 90
  - [ ] HTTPS 使用
  - [ ] 無 console 錯誤
  - [ ] 圖片最佳化

- [ ] **SEO**
  - [ ] 分數 ≥ 90
  - [ ] Meta 標籤完整
  - [ ] 語意化 HTML
  - [ ] Sitemap 存在
