# 設計差異分析報告

> **比對基準**：目標網站 https://world.waterballsa.tw/
> **分析日期**：2025-11-22
> **當前實作完成度**：~40%

## 執行摘要

經過與目標網站詳細比對後，發現當前實作的前端雖然具備完整的技術架構（Next.js 14、TypeScript、元件庫），但在**視覺設計**、**佈局結構**、**品牌識別**三個核心面向與目標網站有顯著差異。

### 關鍵發現

1. **色彩方案完全相反**：實作為淺色主題，目標為深色主題
2. **導航結構錯誤**：實作為橫向頂部導航，目標為左側固定垂直側邊欄
3. **缺少品牌元素**：創辦人介紹、折價券橫幅、訂單記錄等特色功能完全缺失
4. **課程卡片設計不符**：缺少擁有狀態、雙 CTA 按鈕、折扣資訊

---

## 1. 色彩方案差異 🎨

### 目標網站設計

| 元素 | 顏色 | 使用方式 |
|------|------|---------|
| 主背景 | `#1A1D2E` | 頁面主要背景色 |
| 次背景 | `#2D3142` | 卡片、側邊欄背景 |
| 主色（金色） | `#FFD700` | 按鈕、標題、重點強調 |
| 主文字 | `#FFFFFF` | 標題與重要文字 |
| 次文字 | `#A0AEC0` | 描述與輔助文字 |
| 邊框 | `#4A5568` | 分隔線、卡片邊框 |

### 當前實作問題

- ❌ **背景顏色**：使用白色/淺灰背景，應為深色 `#1A1D2E`
- ❌ **主色應用錯誤**：金色 `#FFD700` 在 tailwind.config.ts 中定義，但元件中未正確使用
- ❌ **文字顏色**：使用深色文字，應為淺色 `#FFFFFF` / `#A0AEC0`
- ❌ **缺少深色主題**：`globals.css` 未強制套用深色主題

### 修正方案

```css
/* globals.css 需新增 */
:root {
  --background-primary: #1A1D2E;
  --background-secondary: #2D3142;
  --primary: #FFD700;
  --text-primary: #FFFFFF;
  --text-secondary: #A0AEC0;
}

body {
  background-color: var(--background-primary);
  color: var(--text-primary);
}
```
